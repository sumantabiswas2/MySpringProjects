# Eclipse macOS Shortcuts Cheat Sheet

The most critical keyboard shortcut in Eclipse on macOS is **`⌘ + ⇧ + L` (Command + Shift + L)**. Pressing this opens a complete, on-screen index of all available key bindings.

## Essential Shortcuts

### 📝 Code Editing & Generation
* **Content Assist (Autocomplete)**: `^ + Space` (Control + Space)
* **Quick Fix**: `⌘ + 1`
* **Format Code**: `⌘ + ⇧ + F`
* **Organize Imports**: `⌘ + ⇧ + O`
* **Toggle Single-line Comment**: `⌘ + /`
* **Toggle Block Comment**: `⌘ + ⌥ + /`
* **Delete Line**: `⌘ + D`
* **Move Line Up/Down**: `⌥ + ↑` or `⌥ + ↓`
* **System.out.println()**: Type `syso` then press `^ + Space`

### 🔍 Search & Navigation
* **Open Resource (Find File)**: `⌘ + ⇧ + R`
* **Open Type (Find Java Class)**: `⌘ + ⇧ + T`
* **Quick Outline (Methods/Fields)**: `⌘ + O`
* **Go to Definition**: `F3` (or `fn + F5`)
* **Search Workspace for References**: `⌘ + ⇧ + G`
* **Quick Access Search**: `⌘ + 3`

### ⚡ Execution & Refactoring
* **Run Application**: `⌘ + ⇧ + F11`
* **Debug Application**: `⌘ + ⌥ + D` followed by `J`
* **Rename (Refactor)**: `⌘ + ⌥ + R`

---

## ⚠️ Common macOS Conflicts & Fixes

### 1. Autocomplete (`^ + Space`) Conflicts
macOS often intercepts this shortcut to switch keyboard input sources. 
* **Fix**: Open **Mac System Settings > Keyboard > Keyboard Shortcuts > Input Sources** and uncheck "Select the previous input source".

### 2. Function Keys (`F3`, `F11`)
Your Mac might use function keys for hardware actions like brightness or volume.
* **Fix**: Hold down the **`fn`** key while pressing the function shortcut, or toggle the default behavior in **System Settings > Keyboard**.
